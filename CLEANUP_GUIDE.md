# Project Cleanup Guide

This guide lists unnecessary files and dependencies that can be removed from the portfolio project.

## Files That Can Be Removed

### 1. Test Files (if you're not running tests)
```
src/index.test.js
src/App.test.js
src/components/**/*.test.js
src/setupTests.js
```

**Why remove?**
- Test files are only needed if running Jest tests
- These are generated by Create React App by default
- Reduces clutter if not maintaining tests

**How to remove:**
```bash
# Remove individual test files
rm src/index.test.js
rm src/App.test.js
rm src/setupTests.js
# Or remove recursively
rm -r src/**/*.test.js
```

### 2. Development Documentation (Optional)
If using this instead:
```
PORTFOLIO_SETUP.md  # Can remove if using PORTFOLIO_MANAGEMENT.md
```

**Keep:**
- `PORTFOLIO_MANAGEMENT.md` - Main user guide (NEW)
- `ADMIN_GUIDE.md` - Admin documentation
- `README.md` - Project overview

### 3. Environment Configuration Files
```
.env        # Contains API keys (SHOULD NOT be in git!)
```

**Important:** 
- ⚠️ Do NOT commit `.env` to GitHub
- Keep `.env.example` as a template
- Add `.env` to `.gitignore` (already done)

**If you haven't already:**
```bash
# Verify .env is in .gitignore
cat .gitignore

# Should contain: .env
```

## Dependencies That Can Be Removed

### Current Dependencies in package.json:
```json
{
  "axios": "^1.6.0",        // Only used for API calls
  "which": "^3.0.1"         // Only used for system checks
}
```

### Analysis:

#### Axios
- **Keep if:** Using Gemini API or making HTTP requests in Chatbot
- **Remove if:** Not using Chatbot or API features

**Check usage:**
```bash
# Search for axios imports
grep -r "import.*axios" src/
grep -r "from.*axios" src/
```

**If not used, remove:**
```bash
npm uninstall axios
```

#### Which
- **Keep if:** Checking for system executables
- **Remove if:** Not used anywhere

**Check usage:**
```bash
grep -r "import.*which" src/
grep -r "from.*which" src/
```

**If not used, remove:**
```bash
npm uninstall which
```

## Recommended Cleanup

### Minimum Cleanup (Recommended):
1. Remove test files (`*.test.js`)
2. Update documentation (use PORTFOLIO_MANAGEMENT.md)
3. Verify `.env` is not committed

### Optional Cleanup:
1. Remove unused dependencies (axios, which)
2. Remove PORTFOLIO_SETUP.md if using PORTFOLIO_MANAGEMENT.md

### DO NOT Remove:
- ❌ node_modules (regenerated with `npm install`)
- ❌ .env file (needed for local development, but not committed)
- ❌ Any component files in src/components/
- ❌ package-lock.json
