# Quick Start: Add Your First Project 🚀

This is a quick guide to add your first project to the portfolio in 2 minutes.

## Access Portfolio Manager

1. Open your portfolio website
2. Navigate to `/portfolio-manager`
3. Enter password: `portfolio123`

---

## Add a SolidWorks Project (Example)

### Fill in the Form:

| Field | Example Value |
|-------|---------------|
| **Title** | Advanced Mechanical Assembly |
| **Description** | Complex 3D CAD assembly featuring rotating joints, precise constraints, and technical specifications. Designed for manufacturing with full bill of materials. |
| **Icon** | Click 📐 or 🏗️ |
| **Technologies** | SolidWorks, 3D Modeling, CAM |
| **GitHub Link** | (optional) |
| **Demo Link** | (optional) |
| **Image** | Upload your design screenshot/rendering |

### Step-by-Step:

1. **Title** → Type: `Advanced Mechanical Assembly`
2. **Description** → Paste your project description
3. **Icon** → Click the 📐 icon button (it will highlight)
4. **Technologies**:
   - Type: `SolidWorks` → Click **Add**
   - Type: `3D Modeling` → Click **Add**
   - Type: `CAM` → Click **Add**
5. **Upload Image** → Click **Choose File** → Select your design image
6. **Click** → "Add Project" button

### Done! ✅

Your project now appears on your portfolio website instantly.

## Tips ✨

- 📸 **Images**: Upload high-quality PNG/JPG (best size: 800x600px or smaller)
- 🏷️ **Technologies**: Add 3-5 relevant tech tags
- 📝 **Description**: Be specific about what the project does and technologies used
- 🔗 **Links**: Add GitHub or demo links if available
- 😊 **Emoji Icons**: Pick from suggestions or type any emoji

## Troubleshooting

**Image not uploading?**
- Try a smaller file (< 2MB)
- Use JPG or PNG format
- Check browser console for errors (F12)

**Project not showing?**
- Refresh the page
- Check that all required fields are filled
- Verify localStorage is enabled

**Can't login?**
- Password is: `portfolio123`
- Check for typos
- Try refreshing the page
