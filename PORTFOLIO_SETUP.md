## 🚀 Getting Started

### 1. Install Dependencies (Already Done ✓)
```bash
npm install
```

### 2. Set Gemini API Key (Already Done ✓)
Your API key is configured in `.env` file

### 3. Start Development Server
```bash
npm start
```
The app will open at `http://localhost:3000`

### 4. Build for Production
```bash
npm run build
```
